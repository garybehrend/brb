# brb

Hate waiting for slow terminal tasks? Wait no more!

![OS X notification](http://cl.ly/VxU3/Screen%20Shot%202014-06-08%20at%2019.23.07.png)

Requirements
------------

* Mac OS X (>= 10.8)
* node.js

Installation
------------

    $ npm install -g brb

## Usage
	
	$ bundle install
	Installing...
	brb
